# tollge-modules

#### 项目介绍
tollge项目的模块
扩展tollge, 实现各种功能.

#### 安装教程

需要什么module, 请参看module的引入方式.

#### 已经实现的module

|组|具体实现|
|-|-|
|auth 鉴权|localstorge 本地存储的个性实现|
|data 数据源|data-hikari 基于hikari连接的个性实现|
|job 任务| 暂无|
|oss 对象存储|oss-qiniu 七牛对象存储|
|sms 短信|sms-dayu 大鱼短信|
|web 网站服务|web-http http服务|
|wechat 微信|wechat-gzh 公众号实现|
